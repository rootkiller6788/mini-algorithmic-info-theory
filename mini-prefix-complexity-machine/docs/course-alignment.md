# Course Alignment — Mini Prefix Complexity Machine

## Nine-School Mapping

### MIT
- **6.841 Advanced Complexity Theory** §4-5: Kolmogorov complexity, prefix complexity, algorithmic information
- **6.045/18.400 Automata, Computability, Complexity**: Self-delimiting codes, universal machines

### Stanford
- **CS254 Computational Complexity** §4: Algorithmic information theory, Solomonoff induction
- **EE376A Information Theory**: Prefix codes, Kraft inequality, Huffman coding

### Berkeley
- **CS172 Computability & Complexity**: Kolmogorov complexity fundamentals
- **CS278 Computational Complexity**: Algorithmic randomness, Chaitin's Omega
- **EE229A Information Theory**: Source coding, prefix codes

### CMU
- **15-455 Undergraduate Complexity Theory**: Basic Kolmogorov complexity
- **15-855 Graduate Complexity Theory**: Algorithmic information, Solomonoff prior

### Princeton
- **COS 522 Computational Complexity**: Prefix complexity, algorithmic randomness
- **COS 551 Advanced Complexity**: Levin search, universal distribution

### Caltech
- **CS 151 Complexity Theory**: Kolmogorov complexity and physics connections
- **CS 154 Limits of Computation**: Chaitin's Omega, algorithmic information

### Cambridge
- **Part II Complexity Theory**: Kolmogorov complexity, prefix machines
- **Part III Advanced Complexity**: Algorithmic information theory, monotone complexity

### Oxford
- **Computational Complexity**: Prefix complexity and randomness
- **Advanced Complexity Theory**: Algorithmic mutual information, Solomonoff induction

### ETH
- **263-4650 Advanced Complexity**: Algorithmic information, Kolmogorov complexity
- **252-0400 Logic & Computation**: Formal treatment of prefix machines
