# Course Tree: Prerequisites

## This module depends on:
1. Basic probability theory (discrete distributions, Bayes rule)
2. Turing machines (universal computation)
3. Basic information theory (Shannon entropy, Kraft inequality)
4. Asymptotic notation (O, Omega, Theta)

## This module is a prerequisite for:
1. AIXI / Universal Artificial Intelligence
2. Algorithmic learning theory
3. Minimum Description Length (MDL) principle
4. Normalized Compression Distance applications
5. Kolmogorov complexity-based cryptography

## Internal dependency order:
1. Binary string operations -> Program representation
2. Universal machine U -> Program execution
3. Program enumeration -> Dovetailing -> M(x) computation
4. M(x) -> Prediction -> Applications
5. M(x) + K(x) -> Coding Theorem verification
